Marks

Areas Mark: States within the US

Channels

Color

  • Represents attribute of Color (Hue); Red and Blue.
  • This is a good mapping because it very clearly distinguishes the respective political stance of each state.

Size

  • Represents attribute of type Size (Area).
  • This is a bad mapping because the physical Size of each state does not affect the number of electoral votes.

Colormap(s)

Colormap: What 2016 would look like if just women voted

  • Represents attribute of type Divergent Colormaps.
  • This is a good mapping because the two colors represent extremes of a range of political stances.

Marks

Points Mark: Balls (Points) on Graph

Channels

Color

  • Represents attribute of Color (Saturation): the points exist on a pink to orange spectrum.
  • This is a good mapping because it very clearly distinguishes the range of scoring values through the color saturation attribute.

Position

  • Represents attribute of Position (Horizontal and Vertical): the points exist can range from left to right as well as up and down.
  • This is a good mapping because it visualizes the optimal metric for two variables: launch angle and exit velocity. This provides for a more insightful visualization.

Colormap(s)

Colormap: Scoring value (LWTS) of batted balls based on launch angle and speed off the bat, 2015 MLB

  • Represents attribute of type Sequential Colormaps.
  • This is a good mapping because the sequential color scheme indicates difference in the amount of phenomenon which is relevant to the corresponding scoring value that exists on a spectrum.

Marks

Points Mark: Prices (Points) on Graph

Areas Mark: Averages (Boxes)

Channels

Color

  • Represents attribute of Color (Hue): individual and independent colors.
  • This is a good mapping because it clearly seperates each category into its own color. The Hue (color) was the right selection because there is not relationship between the brands.

Position

  • Represents attribute of Position (Vertical): the points can move up and down.
  • This is a good mapping because it is a very intuitive concept that as the price increases, the points will be plotted higher on the y-axis.

Colormap(s)

Colormap: Price Distribution of Popular Smart Device Brands

  • Represents atrtibute of type Categorical Colormaps.
  • This is a good mapping because the visualization is split into categories with distinct colors. A categorical colormap was the right choice for this graph.
